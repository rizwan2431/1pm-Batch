2. c:\users\india\mvnpractice\hello-world\mvn clean

    mvn clean command will clean the project directory for any previous temporary build files. 
    Since the project is all new, there will be no previous build files to delete. 
    The command will thus succeed. 

3. c:\users\india\mvnpractice\hello-world\ mvn package
	mvn package command instructs Maven to run the package build phase which is part of the default build life cycle.
	Maven creates a target subdirectory inside the project root directory. Inside the target directory you will find the finished JAR/war/ear file, 
               as well as lots of temporary files (e.g. a classes directory containing all the compiled classes).
